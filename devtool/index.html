# 云叩开发者工具 Stable Build (1.2.0)

用于调试云函数和云叩插件

Windows 64下载 (opens new window) 、 MacOS下载 (opens new window)

从1.2.0开始，开发者工具针对企微环境做了优化。支持打开企微移动端的环境。使用方法：

  1. 下载安装开发者工具1.2.0版本
  2. 打开开发者工具
  3. 将链接https://open.weixin.qq.com/connect/oauth2/authorize?appid=wwb2ec7f8205e6c33d&redirect_uri=xbbdev://appfront.xbongbong.com/pro/index.html&response_type=code&scope=SCOPE&state=STATE#wechat_redirect复制到电脑端的企业微信中打开。
  4. mac端允许打开应用后将直接登录成功，windows端如出现打不开页面可将链接复制到浏览器打开，并允许打开开发者工具
