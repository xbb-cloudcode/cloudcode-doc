# 使用CLI

  1. 安装CLI
npm i yunkou-cli -g
  1. 初始化项目
yunkou init

技术类型介绍:

  • vue
    基于vue2的模板，预先配置好webpack打包工具，分为携带router和不携带router的版本。
    • vue-with-router 带router版本
    • vue-without-router 不带router版本
  • react
    模板待添加
  • 其他
    普通HTML模板，不使用打包工具，但为了本地调试方便，预先安装了webpack-dev-server

以vue不带router模板为例: (opens new window)

  1. 启动项目
npm run serve # 会默认监听本机的 3334 端口
  1. 在沙箱环境中调试
  • 打开沙箱环境调试地址http://ptf6web.xbongbong.com/#/cloudcode/test
